# Interrupt-Driven-Inventory-System

Interrupt-driven AVR assembly inventory management system integrating a 4×4 keypad, barcode scanner (UART), LCD interface, and wireless transmitter. The system captures item quantities and barcodes, buffers the data in SRAM, displays it on the LCD, and sends formatted inventory updates wirelessly via UART.
